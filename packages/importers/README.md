# @oser/importers

Import pipeline contracts and initial importers for OSER.

Importers convert external source formats into the OSER document model. They do not render final HTML, PDF, EPUB, or paged output.

Current scope:

- shared import result types
- import warnings
- import assets
- TXT importer
- Markdown importer

Planned future formats:

- HTML
- DOCX
- RTF
